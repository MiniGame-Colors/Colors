## 版本说明

+ 分支(Branch)
  + 每个程序员都有自己的分支
  + develop为开发中的最新分支
  + master为稳定分支

+ 迭代
  + 将按照`目标->成果->修改的新目标`的循环来迭代开发
  + 迭代周期以周为单位
